# Flash Sale TV Display

Aplikasi web statis untuk display jadwal Flash Sale di TV toko.

## Fitur

- UI rasio 16:9 untuk TV.
- Countdown real-time untuk promo yang sedang berlangsung.
- Coming soon untuk promo berikutnya.
- Jadwal harian otomatis berubah status.
- Tidak perlu backend.

## Deploy ke Vercel

1. Upload folder ini ke GitHub.
2. Import repository di Vercel.
3. Framework preset: `Other`.
4. Build command kosongkan.
5. Output directory kosongkan atau isi `.`.

## Ubah Jadwal

Edit data `schedule` di `app.js`.
